# Demos

This folder contains short, runnable examples in English.
Configure defaults in `demos/demo_config.py` or via environment variables.

01_api_server_demo.py
- Starts the FastAPI server with API key auth.

02_tool_policy_demo.py
- Demonstrates tool allowlist/denylist enforcement.

03_workflow_async_demo.py
- Runs a small workflow with non-blocking agent calls.

04_graph_memory_demo.py
- Extracts graph triplets and stores them.

05_hierarchical_memory_demo.py
- Adds interactions to hierarchical memory and retrieves context.
